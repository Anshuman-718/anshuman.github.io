# Anshuman Maurya — Portfolio

Premium responsive portfolio with PCB project galleries and lightbox.

## Preview

```powershell
Set-Location "c:\Users\anshu\Documents\Cursor\portfolio"
npx --yes serve .
```

Or open `index.html` directly in a browser.

## Add more media later

Drop images into `assets/` and add thumbnails to the relevant `.gallery-thumbs` block in `index.html`.
